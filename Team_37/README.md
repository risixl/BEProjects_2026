## Project Description
This project implements a semi-automated system for evaluating handwritten
answer scripts using OCR and AI-based text analysis. The system extracts text,
compares it with model answers, and assists evaluators by generating scores
and feedback, significantly reducing manual effort.

## Key Features
- Handwritten text extraction using OCR
- Automated comparison with model answers
- AI-assisted scoring and feedback
- Web-based interface for evaluators
