Project Overview
This project focuses on **monitoring and early risk assessment of natural disasters**
using Machine Learning and Deep Learning techniques.  
The system is designed to analyze environmental and image-based data to detect
potential disaster patterns and provide meaningful risk insights.

The application is deployed as a web service and supports real-time inference
through a lightweight API and web interface.
