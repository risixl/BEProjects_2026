# SMART TOURISM GUIDE

Smart Tourism Guide is a full-stack web application that generates **personalized, day-wise travel itineraries** using **Google Gemini AI**.  
The platform supports **user authentication, authorization, user profiles, audio itinerary narration, and collaborative trip planning**, enabling users to plan trips together in a secure and interactive way.

---

## Features
- AI-generated multi-day travel itineraries  
- User authentication and authorization  
- User profiles with saved itineraries  
- **Audio narration of itineraries (Text-to-Speech)**  
- **Collaborative trip planning (shared itineraries with multiple users)**  
- Budget-aware and interest-based recommendations  
- Structured JSON output with fallback handling  
- Fast itinerary generation using optimized prompts  
- Clean, responsive UI  
- Secure API handling with environment variables  
- Deployed and accessible online  

---

## Tech Stack

### Frontend
- Next.js 14+
- TypeScript
- Tailwind CSS

### Backend / APIs
- Next.js API Routes
- Node.js
- Gemini API (Google Generative AI)

### Database & Authentication
- MongoDB (user profiles, shared itineraries, collaboration data)
- Authentication & Authorization (JWT / session-based)

### Audio & Collaboration
- Text-to-Speech (TTS) for itinerary audio narration
- Role-based access for collaborative trip planning
